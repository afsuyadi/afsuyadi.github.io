--- title: "Forecasting and Optimization of Bioenergy Crop in India" excerpt: "A project that utilize Machine Learning with Bioenergy Crops " collection: portfolio --- A project that uses Machine Learning to forecast and optimize bioenergy crop's production in India. The project is divided into three section :
(a) The use of geology location, rainfall and elevation data to find which feature is the most significant
(b) Training and validating data in order to find the most efficient model
(c) Forecasting and Optimizing the bioenergy crop using the most efficient model
